---
description: Synchronize documentation with code changes
argument-hint: [--mode=sync|update|check] [--docs=path]
---

Automatically keep documentation in sync with code changes.

**Modes:**

- `sync` - Full synchronization
- `update` - Update specific docs
- `check` - Check for inconsistencies

**Features:**

- JSDoc generation
- README updates
- API documentation
- Changelog maintenance

**Usage:**
*start-doc-maintainer --mode=sync
*start-doc-maintainer --mode=update --docs="README.md"
