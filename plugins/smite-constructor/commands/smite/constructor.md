Use the smite-constructor skill for implementation and coding tasks.

**Tasks:**
- `feature` - Implement new feature
- `refactor` - Refactor existing code
- `fix` - Fix bugs and issues

**Complexity:**
- `simple` - Straightforward implementation
- `medium` - Requires some design decisions
- `complex` - Multi-component, architecturally significant

**Output:** Production-ready code with tests

**Usage:**
/smite:constructor --task=feature --complexity=medium
/smite:constructor --task=fix --file="src/components/Header.tsx"
