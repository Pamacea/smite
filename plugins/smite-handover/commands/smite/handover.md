Use the smite-handover skill for documentation and knowledge transfer.

**Modes:**
- `generate` - Create new documentation
- `update` - Update existing docs
- `review` - Review documentation completeness

**Targets:**
- `README` - Project documentation
- `API` - API endpoint docs
- `JSDoc` - Inline code documentation
- `onboarding` - Developer onboarding guide

**Output:** Comprehensive documentation package

**Usage:**
/smite:handover --mode=generate --target=README
/smite:handover --mode=update --target=JSDoc
